# sails-async

These are utilities to enable async/promise methods for controllers and policies

## Guide
Generally, just import `method` or `policy`. Method converts a single controller method, and policy converts a policy function

## API

### Asyncification
#### `respond`
#### `method`
#### `policy`

### Custom Response
#### `send`
#### `fail`
#### `success`
